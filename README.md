# Pole-learning
Learning Pole Structures of Hadronic States using Predictive Uncertainty Estimation

You can find the preprint for this work on arXiv.

## usage
The code consists of: 
- a package in `./src/utils_ml`, containing the core functions
- a series of scripts in `./scripts` which trains the machine learning models
- a series of notebooks in `./notebooks` which vizualizes the results

If you want to test the code yourself
1. install the pole-learning package by running `pip install .`
2. run the desired script in `./scripts`
3. run the notebooks to reproduce the plots (additionally requires jupyter and matplotlib)
